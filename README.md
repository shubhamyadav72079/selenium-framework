# Selenium Automation Framework

## Tech Stack
- Java
- Selenium WebDriver
- TestNG
- Maven
- Extent Reports

## Project Structure
- Page Object Model (POM)
- Test Components
- Utilities
- Test Suites (XML)

## Features
- UI Automation
- Data-driven testing
- Modular framework design
- Reporting with Extent Reports

## How to Run
1. Clone the repository
2. Open in IntelliJ / Eclipse
3. Run TestNG XML files from `testSuits`
4. View reports in generated report folder
